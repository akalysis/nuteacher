# Digital Garden

Welcome to my personal Digital Garden. This repository contains the source code for a modern, dynamic, and premium web application built with **Next.js**.

## Overview
This project was completely re-architected to move away from a basic static site to a high-performance Next.js application. It features:
- A custom, glassmorphic expanding sidebar for effortless navigation across topics like the *Newcastle 85+ Study* and *Health Expectancies*.
- A highly readable, premium layout designed to showcase research and methodology.
- Support for MDX, allowing seamless integration of Markdown content with interactive React components.

## Development
To run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Architecture
- Framework: [Next.js](https://nextjs.org) (App Router)
- Language: TypeScript
- Styling: Custom Vanilla CSS (no Tailwind) to ensure maximum flexibility and aesthetic control.
